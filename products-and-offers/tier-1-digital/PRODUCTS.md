# Tier 1 Digital Products ($19-99)

**Audience:** The Curious Learner — new developers, vibe coders, career changers, tech-curious professionals, students.

**Principle:** These products are tangible extensions of free content. A viewer watches a video, wants something they can hold onto and reference, and the product fulfills that. The price point is low enough to be an impulse buy, high enough to feel like a real product.

---

## 1. The Prompt Architecture Library ($29-49)

Not just a list of prompts. This is the reasoning behind each one — why it works, what it's doing mechanically, and how to modify it for any model. The difference between this and a free prompt list is the same difference between a recipe and understanding how cooking works.

**What's included:**
- Prompts organized by use case and reasoning pattern
- Annotations explaining the mechanics (why each structural choice matters)
- Modification guides for adapting prompts across models
- Framework for writing original prompts using the same principles

**Pricing:** $29-49

**Content Connection:** Videos about prompt architecture (Pillar 3 — "The Prompt Architect") naturally lead here. A viewer who watches a breakdown of why a prompt works will want the full library.

## 2. The Computational Orchestration Cheat Sheet ($19-29)

A visual decision framework. "I have this problem. Should I use AI, traditional code, a database, or a human?" with decision trees and real examples. This is the 60/30/10 framework turned into something you can pin to your wall or pull up on a second monitor.

**What's included:**
- Decision tree for choosing the right tool for any problem
- Real-world examples mapped to each decision path
- The 60/30/10 framework in visual reference format
- Quick-reference guide for common integration patterns

**Pricing:** $19-29

**Content Connection:** Videos about the 60/30/10 framework (Pillar 2 — "The Right Tool for the Job") naturally lead here. The cheat sheet is what viewers wish they had while watching the video.

## 3. The Abstraction Layer Map ($19)

A beautiful visual reference (poster-quality) showing the layers from electrons to AI, with annotations about what each layer does and when you'd work at each level. This makes the invisible architecture of computing visible and intuitive.

**What's included:**
- Full visual map from hardware to AI (electrons, transistors, logic gates, machine code, OS, applications, AI)
- Annotations at each layer explaining what it does and why it exists
- "You are here" markers for common developer activities
- Historical context showing how each layer emerged

**Pricing:** $19

**Content Connection:** "Layers Beneath" videos (Pillar 1 — "The Layers Beneath") naturally lead here. Viewers who love the "beneath the surface" explanations want a reference they can study and share.

## 4. Code Template Packs ($49-99)

Pre-built, well-documented code templates for common AI integration patterns. Not tool-specific, but pattern-specific. These templates teach by example — each one is a working implementation of an architectural principle from the content.

**What's included:**
- Templates organized by integration pattern (not by tool or language)
- Heavy documentation explaining why each architectural choice was made
- Modification guides for adapting to specific tech stacks
- Anti-pattern examples showing what not to do and why

**Pricing:** $49-99

**Content Connection:** "Builder's Architecture" videos (Pillar 4 — "The Builder's Architecture") naturally lead here. Viewers who are ready to build want starting points that embody the principles they've learned.
