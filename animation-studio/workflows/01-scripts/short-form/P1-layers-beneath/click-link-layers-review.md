## SCRIPT

**HOOK (0-5s):**
If you've ever clicked a link and thought that was one simple action... it's not.

**BUILD (5-30s):**
That one click triggers roughly eight different systems before you see the new page.

Your browser checks its cache. If nothing's there, it asks DNS to translate the domain name into an IP address. Then it opens a TCP connection. Negotiates TLS encryption. Sends an HTTP request. The server processes it. Sends back HTML. Your browser parses it, requests CSS and JavaScript, renders the layout, executes the scripts, and paints pixels.

Eight separate systems, each solving one specific problem. Cache for speed. DNS for addressing. TLS for security. HTTP for communication. Each layer built by someone who said "this part is too hard, let me make it simpler."

**LAND (30-35s):**
Every click is a conversation between eight systems you never see. And each one is an abstraction layer making the internet feel instant.
