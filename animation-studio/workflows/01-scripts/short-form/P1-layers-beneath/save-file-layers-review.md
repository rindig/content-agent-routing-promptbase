## SCRIPT

**HOOK (0-5s):**
If you've ever hit "Save" and thought your file just went to a folder... it didn't.

**BUILD (5-30s):**
That file passes through six different layers before it touches the disk.

Your application formats the data. The operating system translates it into filesystem operations. The filesystem decides where on the disk to write it. The disk controller translates that into electrical signals. Those signals move the read/write head. The head magnetizes tiny regions of a spinning metal platter.

Or if it's an SSD, it traps electrons in microscopic cells instead.

Six layers between "Save" and physics. Each one built so you never have to think about magnetism or electrons. You just think "Save."

**LAND (30-35s):**
Every layer is someone saying "the layer below this is too complicated, I'll handle it for you." That's what abstraction is. And it goes all the way down.
