# Topic Engine

## What This Workspace Is

The topic pipeline. Ideas get captured, formalized with IDs, and tracked through production here.

---

## Files

| File | What It Contains |
|------|-----------------|
| `topic-bank.md` | All topics (T-01 to T-50 initial, T-51+ for new). Status, pillar, funnel position per topic. |
| `brainstorm.md` | Raw idea capture. Unstructured. Ideas graduate to topic-bank when formalized. |

---

## The Process

1. **Check the bank** — scan `topic-bank.md` for available topics before creating new ones
2. **Capture raw ideas** in `brainstorm.md` — no structure required
3. **Formalize** — when an idea is ready, assign it a T-number and add it to `topic-bank.md` with pillar, funnel position, and status
4. **Update status** as topics move through the pipeline (idea → scripted → produced → published)
