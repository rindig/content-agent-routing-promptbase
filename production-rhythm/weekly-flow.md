# Weekly Production Flow

This is a flexible guide, not a rigid schedule. The daily structure exists to batch similar work together and reduce context-switching. If a week looks different, that's fine. The default rhythm is here so you don't have to decide what to work on — you just follow the pattern.

---

## The Weekly Rhythm

### Monday: Script Batch

Write 5-7 scripts for the week. Each script should be 80-180 words.

**Why Monday:** Writing is the highest-leverage creative work. Doing it first, when mental energy is freshest, means better hooks and tighter scripts. Batching all scripts in one session keeps you in the same headspace — you start seeing connections between topics, and the quality of script 5 is better because you've already warmed up on scripts 1-4.

**How:**
- Pull from topic-engine/topic-bank.md for what to write about next
- Use the hook templates from script-lab/hook-system.md
- Map each script to a pillar, funnel position, and platform
- Tag each script with its hook type (for analytics tracking later)

---

### Tuesday-Wednesday: Animation Batch

Run scripts through Remotion + Claude Code. You can generate 30 animations in under an hour. Batch all weekly animations in one session.

**Why two days:** This gives flexibility. Some weeks you'll knock out all animations Tuesday morning. Other weeks you'll spread it across two days. The key is that scripts are already done — you're not writing and animating at the same time.

**Why this matters:** This is the production advantage that's nearly impossible to replicate. Most creators spend hours per video on motion graphics. The Remotion + Claude Code pipeline turns animation from a bottleneck into a batch process. Protect this workflow.

---

### Thursday: Voice + Edit

Record all voiceovers in one session. Assemble in CapCut (voiceover + animation + captions + music bed).

**Why batch voiceovers:** Recording all voiceovers in a single session means consistent audio quality — same room tone, same mic position, same energy level. Listeners notice when audio quality shifts between videos, even if they can't articulate why.

**Why Thursday:** This is the bottleneck of the entire pipeline. Animation is automated, but voiceover and editing still require hands-on time. Placing it on Thursday means you have a full day without other production tasks competing for attention. Protect this time.

---

### Friday: Schedule + Distribute

Upload and schedule across platforms. Write LinkedIn text posts to accompany video drops. Queue everything.

**Why Friday:** Distribution is administrative, not creative. It requires attention to detail (right format, right copy, right scheduling time) but not deep creative energy. Doing it Friday means you enter the weekend with everything queued — nothing hanging over you.

**How:**
- Reference platform-playbook/ for platform-specific optimization
- Write LinkedIn posts that complement (not duplicate) the video content
- Schedule across platforms using each platform's native scheduler or a scheduling tool
- Double-check format specs match each platform's requirements

---

### Weekend: Community + Ideas

Respond to comments. Note which questions people ask. Monitor analytics. Identify what's resonating.

**Why weekends:** This is low-effort, high-signal work. Scrolling through comments and checking analytics doesn't require deep focus, but the insights are valuable. Questions people ask in comments become next week's topics. Videos that outperform tell you what to make more of.

**How:**
- Respond to comments (builds community, signals to algorithms)
- Note recurring questions — add them to topic-engine/brainstorm.md
- Quick analytics check: anything significantly outperforming or underperforming?
- Capture any new topic ideas that come up organically

---

## Volume Targets

- **5-7 shorts per week** across platforms
- **Content mix:**
  - 3-4 fully animated explainers (hero content — these carry the brand)
  - 1-2 simpler "text animation" or "diagram build" pieces (lighter production, still valuable)
  - 1 counter-position or honest take (can be simpler visually — the strength is in the perspective)
- **3-4 LinkedIn posts per week** (text posts that complement or stand alone from video content)
- **Content ratio by funnel position:**
  - 60-70% top of funnel (broad reach, discovery content)
  - 25-30% middle of funnel (deeper exploration, building trust)
  - 5-10% bottom of funnel (direct tie to products or services)

---

## Production Pipeline

How a script becomes published content:

1. **Write the script** (script-lab/) — 80-180 words, mapped to pillar and hook type
2. **Generate animations** in Remotion via Claude Code (separate pipeline folder) — batch process, not one-at-a-time
3. **Record voiceover** — natural voice, calm and measured, not "podcast energy"
4. **Assemble in CapCut** — voiceover + animation + captions + music bed
5. **Export in platform-specific formats** — check format specs below
6. **Schedule and distribute** — reference platform-playbook/ for optimization

---

## Format Specs

| Element | Spec |
|---|---|
| TikTok/Reels length | 30-60 seconds |
| YouTube Shorts length | 60-90 seconds |
| Aspect ratio | Vertical 9:16 |
| Presenter | Voiceover only, no face |
| Animation style | Tight, purposeful — illustrates the concept, not decoration |
| Captions | Burned in (accessibility + silent scrolling) |
| Music | Low music bed, supports but doesn't compete with voiceover |
