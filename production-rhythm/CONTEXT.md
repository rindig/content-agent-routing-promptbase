# Production Rhythm

## What This Workspace Is

The operational hub. Weekly workflow, analytics tracking, and the feedback loop that connects performance data back into content decisions.

| File | What It Contains |

## How to Use This Workspace

- **Planning the week** — read `weekly-flow.md` for the cadence and task flow
- **Reviewing performance** — read `analytics-tracker.md` for metrics and what they mean
- **Feeding insights back** — performance data informs topic selection (`../topic-engine/`) and platform strategy (`../platform-playbook/`)
