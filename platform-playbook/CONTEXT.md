# Platform Playbook

## What This Workspace Is

Distribution strategy. Where content goes, how it's optimized per platform, and how the content funnel drives audience from discovery to conversion.

---

## Files

| File | What It Contains |
|------|-----------------|
| `funnel-map.md` | Content funnel tiers (TOF/MOF/BOF), volume ratios, pillar mapping, offer connection patterns |
| `platform-specs.md` | Per-platform format rules, posting frequency, optimization notes (TikTok/Reels, YouTube, LinkedIn) |

---

## How to Use This Workspace

- **Planning content distribution** — read `funnel-map.md` first for the strategic frame, then `platform-specs.md` for the target platform
- **Optimizing a script for a platform** — read the platform's section in `platform-specs.md`
- **Connecting content to products** — `funnel-map.md` maps content types to funnel stages
